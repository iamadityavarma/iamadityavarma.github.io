COVID-19 Radiography Image Detection Model

This project uses machine learning models to classify radiography images into categories: COVID, Non-COVID, and Normal. The workflow involves data preprocessing, image augmentation, and training using advanced models such as EfficientNet B5, InceptionV3, MobileNetV3, and VGG19. The models are evaluated on test data with performance metrics like precision, recall, and confusion matrices, supported by LIME for interpretability.

Plagiarism Detection using Data Mining, Neural Networks and NLP

This project applies advanced data mining, neural networks, and natural language processing (NLP) techniques to detect plagiarism. It includes preprocessing steps like tokenization and text cleaning, followed by training models such as Bi-Directional RNNs, LSTMs, and CNN-LSTM hybrids using GloVe embeddings. The evaluation phase includes performance metrics, confusion matrices, and visualizations of training accuracy/loss. The model can predict plagiarism in real-time on unseen documents.
